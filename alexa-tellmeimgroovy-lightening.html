  • Uses Account Linking

How Account Linking Works

  • Echo app passes your Amazon ID to our App
  • Register your user ID in app - add your keys
  • We use Spring Security OAuth2 to link your AWS to Web Account
  • Run skill via Grails App to talk to AWS
